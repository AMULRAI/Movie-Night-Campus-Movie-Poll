# 🎬 Movie Night: Campus Movie Poll

## 📌 Overview

**Movie Night: Campus Movie Poll** is a mobile event polling application designed to help college students collaboratively organize campus movie nights.

The platform allows users to suggest movies, participate in real-time voting, and view poll results to determine which movie will be screened. The system improves student engagement and simplifies event coordination through a digital polling and scheduling experience.

---

## 🎯 Objective

To design and develop an interactive polling-based event tool that enables collaborative movie selection and efficient movie night management.

---

## 🚀 Features

### ✅ MVP Features

* Create movie polls
* Suggest movies
* Real-time voting system
* Display poll results

### ⭐ Advanced Features (Future Scope)

* Schedule movie night events
* Genre-based movie filtering
* Group chat for participants
* Post-event history and analytics
* Admin moderation panel
* Seat availability tracking

## ⚙️ Installation & Setup

1. Clone the repository

```
git clone https://github.com/yourusername/movie-night-campus-poll.git
```

2. Navigate to project

```
cd movie-night-campus-poll
```

3. Install dependencies

```
npm install
```

4. Run the application

```
npm start
```
